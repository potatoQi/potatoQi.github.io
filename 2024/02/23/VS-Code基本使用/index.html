VS Code基本使用

发表于 2024-02-23 更新于 2024-02-25 分类于 杂项 阅读次数：
本文字数： 1.1k 阅读时长 ≈ 1 分钟

VS Code作为最常用的IDE，掌握其使用方法以及相关插件十分必要。

而且这玩意还可以通过Github同步配置，所以配置一次，永久享用。

而且VS Code里有终端，于是写项目配合git使用很方便。

快捷键

  • ctrl + ,：设置
  • ctrl + shift + p：命令面板
  • ctrl + p：最近文件列表
  • ctrl + j：切出下面板（用来在代码和终端输入间跳转）

插件

code runner

  • ctrl + alt + n：编译运行程序
  • ctrl + c：退出当前命令（死循环时退出运行）

WSL

  • 直接在本地vscode登入进wsl子系统中的vscode
  • 或者在wsl中输入code .进入vscode界面

Git Graph

  • 装了之后，直接在vscode中就可以查看提交/分支状态
  • 可以查看每次commit的id/author/date/parents/与上一次commit的不同之处
    • 查看与任意一次commit的不同之处：先点一个，再按住ctrl点另一个

Todo Tree

  • TODO表示待办，FIXME表示写了一半还没写完，BUG表示这段程序有问题，HACK表示这段程序可以优化，NOTE表示想法
  • 这个插件除了方便写注释，还可以充当打标签的作用

Bookmarks

  • 打标签，当程序很长的时候，用鼠标跳转很不方便，用书签跳转就很快。

Jupyter

  • 无需安装jupyter notebook即可在vscode实现相同功能
  • Esc + M：markdown模式
  • Esc + Y：代码模式
  • ctrl + enter：运行
  • Esc + D, D：删除该单元
  • Esc + B：在下方添加一个单元
  • Esc + A：在上方添加一个单元
  • Esc + L：显示行号

Markdown All in One & Markdown Preview Enhanced

  • 第一个插件就是markdown插件，第二个是增强功能插件
  • 第二个增强插件很强大，甚至可以渲染出matplotlib代码的图像。
  • ctrl + k, v：打开预览面板
  • ``：行内代码
  • - [x]：任务列表
  • [^1]和[^1]: 配套使用 ：脚注
  • ==xxx==：高亮
  • 渲染plt代码图片：（只需将```python改为```python {cmd=true matplotlib=true}）
    • 这个功能只能在本地写markdo使用，hexo博客渲染不了

Draw.io Integration

  • 配合markdown使用，画流程图

Markdown Table

  • 配合markdown使用，快速生成表格
  • 选中一段文字，右键菜单中选择Markdown Table: Convert TSV to table就能把这段文字变成表格。
  • 插入新的一行：当新行输入|，按tab补全
  • 在某列左/右插入新的一列：选中当前列，右键菜单插入即可
  • 当前列对齐方式：选中当前列，右键菜单选择即可
  • 交换相邻两列：选中当前列，右键菜单选择即可
